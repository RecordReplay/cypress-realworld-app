## Instructions

Getting started

1. `yarn`
1. `yarn build:ci`

Running the tests locally without Replay

1. `yarn start:ci`
1. `yarn cypress:run`

Running the tests locally with Replay

1. `yarn start:ci`
1. `yarn cypress:run:replay`
